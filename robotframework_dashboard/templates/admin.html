Customize View

Admin custom view usage instructions

  • You can set which sections and graphs are hidden by default as admin
  • If a user opens the dashboard and does not yet have any 'hidden settings' the admin config will be used and saved in local storage
  • If a user does already have 'hidden settings' the admin settings are ignored even if they are different
  • To view current 'hidden settings' (both admin and user) you can open 'Developer Tools (F12)' > 'App' > 'Local Storage'
  • Only when refreshing the dashboard page will admin settings be used for above checks
